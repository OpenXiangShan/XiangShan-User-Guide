# 香山用户指南

## 构建

我们使用 Pandoc 和 MkDocs 来构建此文档。

### Pandoc

Pandoc 用于构建 PDF 和单页 HTML 格式的文档。

```bash
# 安装依赖
bash ./utils/dependency.sh

# 执行构建
make
```

### MkDocs

MkDocs 用来构建部署于互联网的静态网站。

```bash
# 安装依赖
pip install -r requirements.txt

# 预览网站
mkdocs serve

# 构建网站
mkdocs build
```
